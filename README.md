# cityselect

## 安装
```
yarn add @fuyangzhijian/vuecityselect
```
## 导入
```
main.js添加
import "@fuyangzhijian/vuecityselect/lib/cityselect.css"
import CitySelect from '@fuyangzhijian/vuecityselect'
```
## 注册
```
main.js添加
Vue.use(CitySelect)
```
